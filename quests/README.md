## 🗺️ Aventures disponibles

### 🩺 [Ausculter la Bête de Métal](nethardware/) — ⭐⭐⭐⭐ Rang S
Une créature mécanique arrive à l'agonie. **Diagnostique** son mal avant qu'elle ne s'éteigne.
> *Projet réel : [nethardware-monitor](https://github.com/LeDeutsch/nethardware-monitor) — supervision temps réel de composants PC.*

### 🈁 [Traduire les Runes d'Aincrad](sao-utils/) — ⭐⭐⭐ Rang A
Un grimoire venu d'un autre monde attend d'être décrypté. **Prouve ta maîtrise de la langue des anciens.**
> *Projet réel : [SAO-Utils-Patch-lang-fr](https://github.com/LeDeutsch/SAO-Utils-Patch-lang-fr) — traduction FR de SAO Utils.*

### 🔐 [Enchanter les Sceaux d'Accès](estiam-rfid/) — ⭐⭐⭐ Rang A
Une porte magique refuse tous les intrus. **Trouve le bon sceau pour l'ouvrir.**
> *Projet réel : [Estiam-RFID](https://github.com/LeDeutsch/Estiam-RFID) — auth par badges RFID.*

### 📚 [Ordonner le Grimoire Unity](unity-hierarchy/) — ⭐⭐ Rang B
Un grimoire chaotique doit être classé. **Fais l'ordre parmi les runes.**
> *Projet réel : [custom-hierarchy-Unity](https://github.com/LeDeutsch/custom-hierarchy-Unity) — plugin Unity.*

### 🎨 [Tisser les Voiles de Style](lulu/) — ⭐⭐ Rang B
Une illusion sans forme cherche son habit. **Aide-la à devenir visible.**
> *Projet réel : [LULU](https://github.com/LeDeutsch/LULU) — expérimentation CSS.*

## 🎖️ Carte d'Aventurier

Chaque quête complétée = un tampon sur ta carte.
Reviens à la Guilde après chaque victoire — l'hôtesse tient les comptes.

- ⬜⬜⬜⬜⬜ **0/5** quêtes accomplies → *Voyageur curieux*
- ⬛⬛⬛⬜⬜ **3/5** → *Compagnon de la Guilde*
- ⬛⬛⬛⬛⬛ **5/5** → *Membre d'Honneur* 🏅 → **quête cachée débloquée**

*(Système d'honneur — les tampons sont dans ton cœur, pas dans une base de données.)*
