# 🔐 Quête : Enchanter les Sceaux d'Accès

**Rang :** ⭐⭐⭐ A · **Type :** Sécurité · **Récompense :** Titre *« Gardien des Portes »*

---

## 📖 Contexte

> *« Une porte magique refuse tous les intrus. Ludwig a forgé les sceaux qui l'ouvrent.
> Sauras-tu reconnaître le bon parmi les faux ? »*

---

## 🚧 Quête en construction

Mini-jeu prévu : **choisir le bon badge parmi 4**. Trois sont mal formés / expirés / mal signés, un seul ouvre la porte. Reveal expliquant l'architecture d'auth.

**Projet réel :** [Estiam-RFID](https://github.com/LeDeutsch/Estiam-RFID) — application web d'authentification par badges RFID.
